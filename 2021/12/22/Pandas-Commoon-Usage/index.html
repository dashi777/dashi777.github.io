    2021年-Python数据分析Pandas

    引言

    Pandas是Python 语言的一个扩展程序库，用于数据分析。是一个功能非常强大的扩展库，提供高性能、易于使用的数据结构和数据分析工具。通过Pandas库，可以非常方便地从SQL数据库、Excel、CSV等数据源读取数据，并且提供SQLlike的数据处理操作，广泛用于学术、金融、统计学等各个数据分析领域。

    本文简单介绍一下我在使用Pandas库的常用操作，除了用于分享外，主要用于方便查找，本文会持续更新，由于本人知识所限，如文中有纰漏，欢迎指出；也欢迎讨论。

    另外：下午介绍的方法/函数并没有将所有的参数进行介绍，如需进一步了解函数的其他用法，或者需要了解Pandas的其他使用函数/方法，建议到Pandas的官方文档进行查询，官方文档有所有函数/方法的记录并配有相关的例子，链接如下：

    Pandas官方文档

    安装

    因为Pandas是Python的一个扩展库，使用的前提是安装相关的库，假设你已经安装了Python，因为Python2已经停止维护，本文全部以Python3为前提进行描述。

    使用pip工具进行安装：

    $ pip install pandas

    安装成功后就可以在Python脚本中引用和使用：

    import pandas
pandas.__version__  # 查看版本

    基础知识

    Series

    是一种类似于一维数组的对象，它由一组数据（各种Numpy数据类型）以及一组与之相关的数据标签（即索引）组成。

    DataFrame

    是一个表格型的数据结构，它含有一组有序的列，每列可以是不同的值类型（数值、字符串、布尔型值）。DataFrame 既有行索引也有列索引，它可以被看做由 Series 组成的字典（共同用一个索引）。

    注：我的数据分析主要使用DataFrame的数据结构，DataFrame可以理解为一个Excel的表格，Index为列 Row为行，DataFrame就是多个Series组合而成的数据结构。

    数据读取

    读取CSV

    函数实例

    #用法
pandas.read_csv(
filepath_or_buffer, sep=NoDefault.no_default, delimiter=None, header='infer', names=NoDefault.no_default, index_col=None, usecols=None, squeeze=False, prefix=NoDefault.no_default, mangle_dupe_cols=True, dtype=None, engine=None, converters=None, true_values=None, false_values=None, skipinitialspace=False, skiprows=None, skipfooter=0, nrows=None, na_values=None, keep_default_na=True, na_filter=True, verbose=False, skip_blank_lines=True, parse_dates=False, infer_datetime_format=False, keep_date_col=False, date_parser=None, dayfirst=False, cache_dates=True, iterator=False, chunksize=None, compression='infer', thousands=None, decimal='.', lineterminator=None, quotechar='"', quoting=0, doublequote=True, escapechar=None, comment=None, encoding=None, encoding_errors='strict', dialect=None, error_bad_lines=None, warn_bad_lines=None, on_bad_lines=None, delim_whitespace=False, low_memory=True, memory_map=False, float_precision=None, storage_options=None
)

    主要参数

    1. 文件 filepath_or_buffer 这是一个默认参数据，没有参数名，不能为空。
    2. 分隔符 sep 字符型，每行数据内容分隔符号，默认是 , 逗号
    3. 表头 header 支持 int, list of int，第几行是表头，默认会自动推断，会把第一行作为表头
    4. 列名 names 列表List 如果文件不包含列名，那么应该设置 header=None，列名列表中不允许有重复值。
    5. 使用部分列 usecols 列表List 选取部分列，使用这个参数可以加快加载速度并降低内存消耗。
    6. 整数N 读取行数 nrows
    7. 编码 encoding 指定字符集类型，通常指定为’utf-8’。
    # str, default None
pd.read_csv('gairuo.csv', encoding='utf8')pd.read_csv("gairuo.csv",encoding="gb2312") 
# 常见中文# 其他常用编码 ISO-8859-1 latin-1 gbk

pd.read_csv('gairuo.csv', encoding='utf8')pd.read_csv("gairuo.csv",encoding="gb2312")

    读取excel

    函数实例

    pandas.read_excel(
io, sheet_name=0, header=0, names=None, index_col=None, usecols=None, squeeze=False, dtype=None, engine=None, converters=None, true_values=None, false_values=None, skiprows=None, nrows=None, na_values=None, keep_default_na=True, na_filter=True, verbose=False, parse_dates=False, date_parser=None, thousands=None, comment=None, skipfooter=0, convert_float=None, mangle_dupe_cols=True, storage_options=None
)

    主要参数

    1. io：字符串，文件的路径对象。
    2. sheet_name：None、string、int、字符串列表或整数列表，默认为0。字符串用于工作表名称,整数用于零索引工作表位置,字符串列表或整数列表用于请求多个工作表，为None时获取所有工作表。
    3. header：指定作为列名的行，默认0，即取第一行的值为列名。数据为列名行以下的数据；若数据不含列名，则设定 header = None。
    4. index_col：指定列为索引列，默认None列（0索引）用作DataFrame的行标签。
    5. usecols：int或list，默认为None。
      • 如果为None则解析所有列
      • 如果为int则表示要解析的最后一列
      • 如果为int列表则表示要解析的列号列表
      • 如果字符串则表示以逗号分隔的Excel列字母和列范围列表（例如“A：E”或“A，C，E：F”）。范围包括双方。
    6. skiprows：省略指定行数的数据,从第一行开始。
    7. dtype:字典类型{‘列名1’:数据类型，‘列名’:数据类型}，设定指定列的数据类型。

    数据预处理

    空值处理

    空值在Pandas里面是空，没有值的意思。不是0或者其他任何数据，Pandas是在Numpy库上的拓展库，因此空值与Numpy的定义是一样的：numpy.nan()就是Pandas的空值，类似于Python中的None

    #例子如下
import pandas as pd
import numpy as np

df = pd.DataFrame({"name": ['Alfred', 'Batman', 'Catwoman'],
                "toy": [np.nan, 'Batmobile', 'Bullwhip'],
                  })

print(df)
'''
       name        toy
0    Alfred        NaN ←这个就是NaN值，是一个值，但是代表这个位置没有任何元素的意思
1    Batman  Batmobile
2  Catwoman   Bullwhip
'''

    空值填充-fillna

    DataFrame.fillna(value=None, method=None, axis=None, inplace=False, limit=None, downcast=None, **kwargs)
#例子
df.fillna(0)              #用实数0填充na，
df.fillna(1)              #可以用其他数字
df.fillna('这是空值')      #或者字符串

    主要参数

    value:需要用什么值去填充缺失值

    axis:确定填充维度，从行开始或是从列开始

    method：{‘backfill’, ‘bfill’, ‘pad’, ‘ffill’, None}, default None ffill:用缺失值前面的一个值代替缺失值，如果axis =1，那么就是横向的前面的值替换后面的缺失值，如果axis=0，那么则是上面的值替换下面的缺失值。backfill/bfill，缺失值后面的一个值代替前面的缺失值。注意这个参数不能与value同时出现

    limit:确定填充的个数，如果limit=2，则只填充两个缺失值。

    判断每列是否有空值

    print(dataset_oracle.isnull().any())

    判断每行是否空值

    print(dataset_oracle.isnull().T.any())

    判断某列是否有空值

    print(dataset_oracle[(dataset_oracle['app_type'].isna())])

    空值统计

    print( df.isna().sum())

    删除空值-dropna

    DataFrame.dropna(axis=0, how='any', thresh=None, subset=None, inplace=False)
#用法举例
df = df.dropna(subset=['label'],axis=0)

    主要参数

    axis: 确定删除的维度 0-index 1-column

    how : all-全部是NA才删 any-只要有NA就全删

    subset: list 指定删除的列，需要用列表形式表示

    删除操作

    删除重复项

    DataFrame.drop_duplicates(subset=None, keep='first', inplace=False, ignore_index=False)
#用法举例
DataFrame.drop_duplicates(subset=['label'],keep='first')

    主要参数

    subset: list 指定删除的列，需要用列表形式表示

    keep: {‘first’, ‘last’, False}, default ‘first’ 删除重复项并保留第一次/最后一次/不保留出现的项

    删除符合条件项

    # 单个条件
df = df.drop(df[<some boolean condition>].index)
df_clear = df.drop(df[df['x']<0.01].index)
# 也可以使用多个条件
df_clear = df.drop(df[(df['x']<0.01) | (df['x']>10)].index) #删除x小于0.01或大于10的行

    删除指定行/列

    DataFrame.drop(labels=None, axis=0, index=None, columns=None, level=None, inplace=False, errors='raise')
# 用法示例--删除指定列
DataFrame.drop(['day','hour','app_type','app_sub_type','appname'],axis=1) # axis= 1 删除列 axis=0（默认）删除行
# 用法示例--删除指定行
DataFrame.drop([0, 1])

    主要参数

    labels:要删除的行列的名字，用列表/字符串(单个label)给定

    axis: 默认为 0，指删除行，因此删除 columns 时要指定 axis=1

    index:直接指定要删除的行

    columns:直接指定要删除的列

    Merge

    函数实例

    （merge 类似 SQL：innerjoin 或者 Excel：VLOOKUP 匹配两个DataFrame指定列的数据）

    DataFrame.merge( left, right, how='inner', on=None, left_on=None, right_on=None, left_index=False, right_index=False, sort=False, suffixes=('_x', '_y'), copy=True, indicator=False, validate=None)
# 用法示例
dataset = pd.merge(DataFrame_left,DataFrame_right,on = "label",how='left')

    主要参数

    1. how : {‘left’, ‘right’, ‘outer’, ‘inner’}, default ‘inner’ 默认为合并两个frame的交集
    2. Type of merge to be performed. 合并类型。
      • left: 仅使用左frame中的键，类似于SQL左外部联接；保留关键顺序
      • right: 仅使用右frame中的键，类似于SQL右外部联接；保留关键顺序。
      • outer: 使用两个frame中键的并集，类似于SQL完全外部联接；按字典顺序对键进行排序。
      • inner: 使用两个frame中关键点的交集，类似于SQL内部联接；保留左键的顺序。
    3. on : label or list 列名或索引 ，必须在两个DataFrame中都能找到。如果on为None且未用 索引 合并，则默认为两个DataFrame中列的交集
    4. left_on : label or list, or array-like 左DataFrame中的列名或索引。也可以是左DataFrame长度的数组或数组列表。
    5. right_on : label or list, or array-like

    Concat

    函数实例

    pandas.concat(objs, axis=0, join='outer', ignore_index=False, keys=None, levels=None, names=None, verify_integrity=False, sort=False, copy=True)
# 使用示例
import pandas as pd 
df1 = pd.DataFrame([['a', 1], ['b', 2]],
                   columns=['letter', 'number'])
'''df1
  letter  number
0      a       1
1      b       2
'''
df2 = pd.DataFrame([['c', 3], ['d', 4]],
                   columns=['letter', 'number'])
'''df2
  letter  number
0      c       3
1      d       4
'''
result = pd.concat([df1,df2]) #按行合并
'''result
  letter  number
0      a       1
1      b       2
0      c       3
1      d       4
'''
result = pd.concat([df1,df2],axis = 1 ) #按列合并
'''result
  letter_x  number_y  letter_y  number_y
0      a       1         c       3
1      b       2         d       4 
'''

    主要参数

    objs：需要连接的对象集合，一般是列表或字典

    axis：连接轴向 0是index 1是column 默认是0

    join：参数为‘outer’或‘inner’，默认是outer outer是取并集，inner是取交集

    数据输出

    输出至CSV

    函数实例

    DataFrame.to_csv(path_or_buf=None, sep=',', na_rep='', float_format=None, columns=None, header=True, index=True, index_label=None, mode='w', encoding=None, compression='infer', quoting=None, quotechar='"', line_terminator=None, chunksize=None, date_format=None, doublequote=True, escapechar=None, decimal='.', errors='strict', storage_options=None)
# 使用示例
DataFrame.to_csv('csv_name.csv',index = 0,encoding ='utf-8') # 以UTF-8编码输出CSV

    主要参数

    filepath_or_buffer: 文件输出路径，默认是Python工作路径所在目录，也可以用绝对路径表示

    sep:指定分隔符。如果不指定参数，则会使用逗号分隔

    header: 是否保留表头，字符串或布尔列表，默认为true

    index: 布尔值，默认为Ture 是否保留列索引

    输出至Excel

    函数实例

    DataFrame.to_excel(excel_writer, sheet_name='Sheet1', na_rep='', float_format=None, columns=None, header=True, index=True, index_label=None, startrow=0, startcol=0, engine=None, merge_cells=True, encoding=None, inf_rep='inf', verbose=True, freeze_panes=None, storage_options=None)
# 使用示例
DataFrame.to_excel('excel_name.xlsx',index = 0)

    主要参数

    excel_writer：文件路径或现有的ExcelWriter

    sheet_name：它是指包含DataFrame的工作表的名称

    header: 是否保留表头，字符串或布尔列表，默认为true

    index: 布尔值，默认为Ture 是否保留列索引

    encoding：这是一个可选参数, 可对生成的excel文件进行编码。仅对于xlwt是必需的。
