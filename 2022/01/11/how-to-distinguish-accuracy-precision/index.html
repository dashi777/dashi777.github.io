浅谈数据分析第二期--召回率、准确率傻傻分不清

2022-01-11- 2022-01-24
2022年--数据分析
Dash

引言

提及机器学习模型，必须绕不过的几个词语是召回率（Recall）、精确率（Precision）、准确率（Accuracy），这期我们谈谈机器学习中的这几个评价指标。

机器学习评价指标

机器学习中有很多指标用来对模型进行的优劣进行评估，通过这些评估指标我们可以量化模型的性能和表现，因此这些评估指标对于机器学习来讲是很重要的知识，下面我们看看主要的评价指标以及分类：

我们主要研究的是分类问题，并且机器学习中目前业界比较成熟的模型都是做成二分类，下面主要围绕二分类模型进行讨论

例子

为了更好理解各项指标的含义，下面先简单的举一个分类问题的例子，后面的解释按照例子的数据进行

分类问题图解

首先，我们有一个数据集，这个数据集是包含了5张男性照片和5张女性照片一共十张照片的合集

然后我们通过机器学习，对数据集进行分类，得到了分类的数据集：

基本概念

那么这里的分类就可以分为如下情况：

实际（男） 实际（女）
预测（男） TP — True Positive：实际为男性，且判断为男性（正确） FP — False Positive：实际为女性，但判断为男性（错误）
预测（女） FN — False Negative：实际为男性，但判断为女性（错误） TN — True Negative：实际为女性，且判断为女性（正确）

这就是机器学习的混淆矩阵，TP、FP、FN、TN就是混淆矩阵的全部情况，想要理解几个单项指标，必须先理解这几个基本概念的含义：按照实际情况与机器预测的情况结合进行分类。

消化完以上的知识后，我们可以开始对几项指标进行解释。

准确率（Accuracy)

计算公式：

Accuracy == (TP+TN)/(TP+TN+FP+FN)

准确率就是机器学习算法模型预测对的，占全体数据集的比例。

虽然准确率可以判断总的正确率，但是在样本不平衡 的情况下，并不能作为很好的指标来衡量结果。举个简单的例子，比如在一个总样本中，正样本占 90%，负样本占 10%，样本是严重不平衡的。对于这种情况，我们只需要将全部样本预测为正样本即可得到 90% 的高准确率，但实际上我们并没有很用心的分类，只是随便无脑一分而已。这就说明了：由于样本不平衡的问题，导致了得到的高准确率结果含有很大的水分。即如果样本不平衡，准确率就会失效。

精准率（Precision）

计算公式：

Precision == (TP)/(TP+FP)

精准率就是预测为正的数据占数据集样本中数据为正的比例

精准率和准确率看上去有些类似，但是完全不同的两个概念。精准率代表对正样本结果中的预测准确程度，而准确率则代表整体的预测准确程度，既包括正样本，也包括负样本。

在实际情况中，我们并不需要准确率非常高，因为我们很多情况下只是关注某一个情况，例如，我们只会关注不满意用户的情况，那么我们就需要不满意用户的精准率高即可。

召回率（Recall）

计算公式

Recall=TP/(TP+FN)

召回率的应用场景：比如拿网贷违约率为例，相对好用户，我们更关心坏用户，不能错放过任何一个坏用户。因为如果我们过多的将坏用户当成好用户，这样后续可能发生的违约金额会远超过好用户偿还的借贷利息金额，造成严重偿失。召回率越高，代表实际坏用户被预测出来的概率越高，它的含义类似：宁可错杀一千，绝不放过一个。

召回率是关注我们预测的涵盖程度，举一个例子：即我们能否最大限度的将整个样本数据集中的不满用户找出来。

这几项指标大概聊到这里，实际使用的情况下，很多会结合多个指标对模型的效果进行评判，这里就涉及综合指标：F1分数，AOC曲线等，这些后面再聊。
